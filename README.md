# 🧋 BobaDrink - Bubble Tea Ordering Website

A modern bubble tea ordering web application built with **React.js**.  
BobaBliss provides a smooth and responsive user experience for browsing drinks, viewing menus, and managing orders.

## ✨ Features

- 🧋 Interactive bubble tea landing page
- 📱 Responsive design for desktop and mobile devices
- 🎨 Custom SVG bubble tea cup illustrations
- 🛒 Shopping cart functionality
- 🍹 Dynamic drink menu
- ⚡ Reusable React components
- 🎭 Smooth UI animations
- 📍 Order tracking interface

## 🚀 Technologies Used

### Frontend
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Vite

### Tools
- npm
- ESLint
- Git & GitHub
